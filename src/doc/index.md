
# Around Sequme's Documentation

## Build And Install

Use `./src/script/get_opam`.

```
get_opam configure
get_opam boot
get_opam dev-setup /path/to/code
```

The simply `opam install hitscore`

To build hitscoreweb one needs also `opam install re eliom`.
